* Github图片不显示
[[https://blog.csdn.net/qq_38232598/article/details/91346392][Ref]]
1. 利用F12查看不显示的图片的url
2. 去IpAddress.com搜一下这个rul的ip
3. 在/etc/host下添加 ip url
4. sudo hostname url // 令刚刚配置的host生效
