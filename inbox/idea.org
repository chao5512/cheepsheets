* 快捷键 
约定 C ：control，s : command,  S : shift 
+ s-A/S-S : 查找，可以查操作可以查类
+ s-F : find everywhere
+ C-d : debug
+ C-r : run
+ s-U : 大小写转换，toggle case
** 大部分与emacs相同
+ C-j ：向下滚动屏幕
+ C-l ：向上滚动屏幕
+ s-= ：展开代码块
+ s-- : 收起代码块
+ S-s = : 展开所有
+ S-s - ：收起所有
